# School Internship 2026 — NetLicensing Pricing Table Project

> School internship project — building a custom pricing table powered by NetLicensing using AI-assisted coding tools.

---

## About

This repository is the working space for the Labs64 school internship project 2026. It contains the task list, reference materials, and all deliverables produced during the internship.

The project focuses on hands-on experience with modern software development practices: working with a real SaaS licensing platform ([NetLicensing](https://netlicensing.io)), connecting it to an AI assistant via an MCP server, and using AI coding tools to produce polished front-end results.

No prior professional experience is required — the tasks are designed to be completed step by step, and AI tools are there to help along the way.

---

## Goal

This internship project will help you get acquainted with modern AI-driven development of software products. The goal is to create a showcase demonstrating how to combine a pricing table generated by NetLicensing with custom CSS and JavaScript, producing a visually polished and fully functional pricing table that matches an arbitrary customer style. You will use NetLicensing as the licensing backend, set it up (optionally with the help of an MCP server), and create a couple of styled variants of the pricing table using AI-assisted coding tools. By the end of the project you will have a number of working pricing tables, connected to a real licensing and checkout flow.

### 2. Register on NetLicensing and create an account

Create your account on the NetLicensing platform to access the Management Console and API.

- Main site: [netlicensing.io](https://netlicensing.io)
- Getting Started guide: [netlicensing.io/getting-started/](https://netlicensing.io/getting-started/)
- Wiki / documentation: [netlicensing.io/wiki/](https://netlicensing.io/wiki/)

---

### 3. Create an API key for the MCP server

Generate an API key from your account dashboard — this key is required to authenticate the MCP server.

- NetLicensing Management Console (available after registration)
- API documentation: [netlicensing.io/documentation/](https://netlicensing.io/documentation/)

---

### 4. Deploy the MCP server in Docker

Set up the NetLicensing MCP server locally using Docker.

- GitHub repository: [Labs64/NetLicensing-MCP](https://github.com/Labs64/NetLicensing-MCP)
- Setup instructions: [README.md](https://github.com/Labs64/NetLicensing-MCP/blob/master/README.md#quick-start)
- Docker Hub (Labs64): [hub.docker.com/u/labs64](https://hub.docker.com/u/labs64)

- Connect to Claude Desktop on MacOS by adding this config manually to `~/Library/Application Support/Claude/claude_desktop_config.json`:
```
{
  "mcpServers": {
    "netlicensing-local": {
      "command": "docker",
      "args": [
        "run",
        "-i",
        "--rm",
        "-e",
        "NETLICENSING_API_KEY=...",
        "ghcr.io/labs64/netlicensing-mcp:latest"
      ]
    }
  }
}
```

### 5. Find 2–3 websites with different pricing table designs

Research and collect inspiration for pricing table layouts that will guide your design variants. The designs should differ from each other as much as possible — not just visually, but structurally. Aim to cover distinct interaction and layout patterns, for example:

- **Switchable billing tabs** — a toggle or tab bar that switches between billing periods (e.g. monthly / annual) within the same plan view
- **Group tabs** — tabs that switch between audience segments or tiers (e.g. Personal / Enterprise), each showing a different set of plans
- **Highlighted recommended plan** — one plan is visually emphasised as the recommended or most popular option (e.g. with a badge, border, or contrasting colour)

- Use a general search (Google / Pinterest) for "pricing table design inspiration"

### 10. ⭐ Bonus: Integrate the pricing table with NetLicensing checkout

The exported pricing table HTML already includes a "checkout" button — wire it up so it correctly redirects the user to the NetLicensing checkout flow. This makes the pricing table functional end-to-end, not just a visual mockup.

The NetLicensing JavaScript client can be used as an optional helper for generating shop tokens and building the checkout URL programmatically.

- NetLicensing JavaScript client (optional): [Labs64/NetLicensingClient-javascript](https://github.com/Labs64/NetLicensingClient-javascript)
- NetLicensing Shop / checkout documentation: [netlicensing.io/wiki/netlicensing-shop](https://netlicensing.io/wiki/netlicensing-shop)
